Data Challenge



DATA

The data consists of cell phone traces recorded within areas labeled "Mall G" and "Mall S". There are a total of 9,195 traces in Mall G and 50,294 traces in Mall S, but there can be many traces per user per day.

INSIGHTS

The goal of this challenge is to show a quantitative and descriptive analysis of the cell phone data in order to turn data insights into actionable items, e.g. the managers of Mall G should have fun activities on Saturdays!

VISUALIZATION

A clear representation of the results is key to have impact as a data scientist! For this project, I used Python's matplotlib for simple plots and D3 + javascript libraries for more complex graphs.
